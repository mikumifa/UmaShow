## 概览
UmaShow 让训练、赛事查询与记录复盘集中在一个窗口完成，无需在浏览器和表格工具间反复切换。内置 UMDB 数据解析，保障字段准确且可离线使用，更新数据和版本也只需一键。

## 核心功能
- **实时育成监控**：随时查看体力、能力值成长、训练与事件列表，支持窗口置顶以便同屏操作。
- **比赛情报浏览**：按距离、场地、日期筛选，点击即可查看奖励与适性，表格支持排序与高亮。
- **育成复盘**：历史记录集中展示，便于回顾选择与结果；支持导出 CSV 分享或分析。
- **数据联动**：内置 UMDB 协议解析，配合 protobuf 确保数据与游戏版本保持一致。
- **舒适体验**：深浅色主题、中文/日文语言切换与键盘快捷键（`Ctrl/Cmd + F` 聚焦搜索框，`Ctrl/Cmd + [`/`]` 页面切换）。

## 快速上手
1. **下载与安装**：在发行页面或 `release` 目录获取对应系统的安装包（macOS/Windows/Linux）。
2. **启动应用**：首次打开会加载内置 UMDB 数据，无需额外配置。
3. **进入育成面板**：默认首页为实时监控，侧边栏可切换到比赛列表或历史记录。
4. **筛选与搜索**：使用顶部搜索栏与条件筛选快速定位比赛、训练或事件。
5. **数据刷新**：在设置页点击“更新数据”即可同步最新 UMDB 信息。

## 典型使用场景
- **育成进行中**：开启窗口置顶，在训练界面随时查看体力与事件提示，辅助决策。 
- **规划比赛路线**：按照距离和场地筛选，比较奖励后加入计划。 
- **复盘与分享**：结束后导出历史记录为 CSV，结合笔记总结下一次策略。 

## 常见问题
- **需要联网吗？** 应用可离线运行，只有更新数据或检查新版本时会访问网络。
- **数据来源是什么？** 由内置 UMDB 协议文件解析生成，字段与游戏版本保持一致。
- **能导出数据吗？** 育成记录支持导出为 CSV，便于备份或分享。

## 开发者指南
> 想要二次开发或自行构建？以下流程可作为快速参考。

- **环境要求**：Node.js 14+、npm 7+；如需重新生成 protobuf 类型，请提前安装 `protoc`。
- **安装依赖并启动开发模式**：
  ```bash
  npm install
  npm start
  ```
- **构建与打包**：
  ```bash
  npm run build
  npm run package
  ```
- **更新 protobuf 类型（可选）**：`npm run proto:gen`
